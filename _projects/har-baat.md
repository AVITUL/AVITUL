An android application that presents conversations in a calander view. This application is written to present texts from multiple messengers in the same conversation as long as the contact is same.

The application presents user with a calendar view, where chats from that particular date are fetched and presented in order of time.

A text preprocessing project was also made, to return .html files for chats on a particular date.
